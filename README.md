# VaporFi Columbus - Flavor Picker

Are you tired of manually typing custom blend flavors in Lightspeed? Do you have difficulty deciphering flavors other employees have entered?

The VaporFi Flavor Picker is your answer!

- Easily create custom blend flavors with proper formatting to paste directly into Lightspeed
- Choose from named blends or submit suggestions for your own named blends
- Print labels for custom blend bottles
